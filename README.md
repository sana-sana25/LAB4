# FragmentsLab - LAB4

##  Objectif
Ce projet a pour objectif de comprendre et manipuler les **Fragments Android** :

- Création de fragments dynamiques
- Navigation entre fragments
- Gestion des événements (boutons, SeekBar)
- Gestion de l’état (rotation, sauvegarde)
- Compréhension du cycle de vie des fragments

---

##  Structure du projet

- **MainActivity**
  - Gère l'affichage des fragments
  - Utilise FragmentManager et FragmentTransaction

- **FragmentOne**
  - Affiche un texte + bouton
  - Permet d’afficher un message “Bonjour”

- **FragmentTwo**
  - Contient un SeekBar
  - Affiche dynamiquement la valeur

---

##  Fonctionnalités

✔ Navigation entre deux fragments  
✔ Mise à jour dynamique du contenu  
✔ Gestion du bouton "Back"  
✔ Interface moderne   
✔ Interaction utilisateur (clic + slider)
